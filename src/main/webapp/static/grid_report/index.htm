一、各种格式报表演示

下面演示各种格式的报表，Grid++Report报表展现分两种模式：报表打印展现与报表查询展现。点击报表名称列下的链接文字进入对应报表的打印展现模式。点击图标[查询]列下的链接图标进入对应报表的查询展现模式。报表查询展现模式一般用于明细表格类的报表展现，以连续不分页的方式查阅报表数据。
